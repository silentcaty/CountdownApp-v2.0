CountdownApp v2.0 倒计时提醒工具
==================================

【简介】
-------
这是一个基于 Python 编写的图形化倒计时工具，适合用于学习、专注工作、番茄钟提醒等场景。

本程序可设置总倒计时时长，随机休息间隔，每次休息时长，并在提醒时弹出黑屏窗口和播放音效。
支持最小化到系统托盘、上传自定义音频、自动保存配置等功能。


【主要功能】
-----------
1. 自定义倒计时时长（单位：分钟）
2. 自定义随机提醒间隔（最小值与最大值）
3. 设置每次休息提醒时长（单位：秒）
4. 支持选择 4 个内置提示音或上传自定义音频（支持 wav/mp3）
5. 每次提醒以全屏黑底弹窗显示倒计时，同时播放提示音
6. 支持最小化到托盘后台运行
7. “Esc” 可退出弹窗倒计时


【文件说明】
-----------
countdown_app.exe        ← 程序主文件，双击运行
README.txt               ← 当前说明文档
settings.ini             ← 首次运行后自动生成的配置文件（保存自定义音频路径等）

【使用说明】
-----------
1. 双击 countdown_app.exe 运行程序
2. 设置总倒计时时长、随机间隔、每次休息时长
3. 选择提示音，或点击“上传音频”选择自定义文件
4. 点击“开始倒计时”启动程序
5. 倒计时运行过程中将定时弹出全屏提醒窗口并播放声音
6. 可点击“最小化到托盘”后台运行
7. 程序结束后自动弹出 20 分钟全屏倒计时
8. 可随时点击“停止倒计时”或“退出程序”


【常见问题】
-----------
Q: 打开后为什么没声音？
A: 请确保音量开启，并选择了正确的音效文件

Q: 全屏提醒怎么退出？
A: 按下键盘上的 Esc 键即可关闭当前提醒弹窗（有时需要鼠标点击一下，选择当前窗口再按Esc）。

Q: 为什么提示风险或无法运行？
A: 因程序未签名，部分杀毒软件可能误报，选择运行。
